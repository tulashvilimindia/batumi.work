## Accessing the Website

### URLs

| Language | URL |
|----------|-----|
| Georgian (Default) | `http://your-domain.com/ge/` |
| English | `http://your-domain.com/en/` |

The website defaults to Georgian. You can switch languages at any time using the language toggle.

---

## Language Selection

### Switching Languages

1. Look for the **GE / EN** toggle in the top navigation bar
2. Click on your preferred language
3. The page will reload with content in your selected language

### Language Behavior

- **Georgian (GE)**: All content displays in Georgian
- **English (EN)**: Content displays in English when available, falls back to Georgian otherwise
- Your language preference is saved in your browser

## Browsing Jobs

### Job List Page (Homepage)

The homepage displays all active job postings as cards. Each job card shows:

| Field | Description |
|-------|-------------|
| **Title** | Job position title (in your selected language) |
| **Company** | Hiring company name |
| **Location** | City or region |
| **Category** | Job category (IT, Marketing, etc.) |
| **Published Date** | When the job was posted |
| **Deadline** | Application deadline (if specified) |

### Job Badges

Look for these special badges on job cards:

| Badge | Meaning |
|-------|---------|
| **VIP** | Premium/featured listing |
| **Salary** | Salary information is disclosed |

### Pagination

- Jobs are displayed 20 per page by default
- Use the page numbers at the bottom to navigate
- Use Previous/Next buttons for quick navigation

## Searching and Filtering

### Search Box

1. Enter keywords in the search box (e.g., "developer", "manager")
2. Press Enter or click the search button
3. Results will filter to match your keywords

### Filters

| Filter | Description | How to Use |
|--------|-------------|------------|
| **Category** | Filter by job type | Select from dropdown |
| **Region** | Filter by location | Select from dropdown |
| **Has Salary** | Show only jobs with salary info | Toggle on/off |
| **VIP Only** | Show only premium listings | Toggle on/off |

### Combining Filters

- You can combine multiple filters
- All filters work together (AND logic)
- Clear filters by clicking "Reset" or removing individual selections

### URL Sharing

Filters are reflected in the URL, so you can:
- Bookmark filtered searches
- Share filtered results with others
- Return to the same search later

Example: `https://batumi.work/ge/?category=it-programming&has_salary=true`

### Jobs.ge Compatibility

The website uses the same region and category filters as jobs.ge, so jobs are organized exactly as on the original site. This means:
- IT jobs from jobs.ge appear in our IT category
- Jobs from specific regions are correctly filtered by location

## Viewing Job Details

### Job Detail Page

Click on any job card to view full details:

| Section | Content |
|---------|---------|
| **Header** | Job title, company, location |
| **Description** | Full job description and requirements |
| **Salary** | Salary range (if disclosed) |
| **Dates** | Published date and deadline |
| **Source** | Where the job was originally posted |

### Bilingual Content

- If viewing in English but English content isn't available, Georgian content is shown
- Original language is always preserved

### Source Information

Every job shows its source:
- **manual**: Posted directly on this job board
- **jobs.ge**: Aggregated from jobs.ge
- **hr.ge**: Aggregated from hr.ge

A clickable link to the original posting is provided.

## Mobile Experience

### Responsive Design

The website is fully responsive and works on:
- **Desktop**: Full layout with sidebar filters
- **Tablet**: Adapted layout with collapsible filters
- **Mobile**: Single-column layout, tap-friendly buttons

### Mobile Tips

1. **Search**: Tap the search icon to expand the search box
2. **Filters**: Tap "Filters" to expand filter options
3. **Scroll**: Swipe to scroll through job listings
4. **Details**: Tap any job card to view full details

## Tips for Job Seekers

1. **Check Daily**: New jobs are added throughout the day
2. **Use Filters**: Narrow down to relevant opportunities
3. **Save Searches**: Bookmark filtered URLs for quick access
4. **Apply Early**: Some jobs have limited application windows
5. **Check Source**: Visit the original source for most current information

## Troubleshooting

### Common Issues

| Issue | Solution |
|-------|----------|
| Page not loading | Check your internet connection, try refreshing |
| Filters not working | Clear browser cache, try a different browser |
| Wrong language | Click the language toggle (GE/EN) |
| Job details empty | The job may have been removed from source |

### Browser Compatibility

Supported browsers:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
